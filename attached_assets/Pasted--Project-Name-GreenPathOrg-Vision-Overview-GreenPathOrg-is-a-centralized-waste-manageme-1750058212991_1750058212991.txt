🌱 Project Name: GreenPathOrg
 
🌍 Vision & Overview
GreenPathOrg is a centralized waste management and community engagement platform built specifically for Indian rural and semi-urban village ecosystems. It streamlines waste collection, promotes segregation, ensures accountability among stakeholders, and empowers villagers to raise and track issues.
The project includes:
•	A mobile Android app for all user roles: Admin, Manager, Waste Collector, Waste Generator.
•	A web version (browser access) for Admins and Managers for ease of data entry, monitoring, and report generation.
 
🏠 Public Page (Android/Web - before login)
•	Logo & app name with light green theme.
•	Description: “Empowering Clean Villages through Smart Waste Management.”
•	Fields: Login (ID & Password).
•	Button: "Login" (no signups for any role).
 
🔐 Login System
•	Only Admin has hardcoded default credentials.
•	All other users (Manager, Collector, Generator) are added by roles above them.
•	All user IDs follow a structured format:
o	VillageID-M1, VillageID-C1, VillageID-G1, etc.
•	Default password = user ID (changeable after login).
 
👤 User Roles & Their Dashboards
 
🛠️ Admin Dashboard (Mobile + Web)
Purpose: Manages villages and managers. Can monitor everything.
Features:
1.	Add New Village
o	Auto-generates villageID (V001, V002…)
o	Assigns a Manager with default ID/password.
2.	Manage Villages
o	View list of all villages.
o	View Manager performance via feedback reports, issues solved, etc.
3.	Send Announcements
o	Can choose audience:
	All users in all villages.
	Only Managers.
	Only Generators.
o	Banners shown in app.
4.	View Reports
o	All reports across all villages (issue tracking, collection stats, feedback summaries).
5.	Track Issues
o	See raised issues from each village.
o	Track how Managers handle those.
 
👨‍💼 Manager Dashboard (Mobile + Web)
Purpose: Oversees village operations. Creates and monitors collectors and generators.
Features:
1.	Create Waste Generators (Households)
o	Add in bulk or single.
o	Each gets a unique UID + downloadable QR code in 3x4 PDF format.
o	Fields: UID, head of house, phone, house number, village.
2.	Create Waste Collectors
o	Assign each to a specific village and Panchayat.
3.	Manage Users
o	Edit/delete generators or collectors.
o	Reset passwords.
4.	Mark Attendance
o	Daily attendance of collectors.
5.	Send Announcements
o	Can send banners/messages to all users in the village.
o	View message history.
6.	View Issues
o	Raised by generators.
o	Can update status, reply, and mark resolved.
7.	Track Collection Feedback
o	View segregation scores, missed pickups, and photos submitted by collectors.
8.	Reports
o	Weekly/monthly stats on collection, issues, user activity.
 
🚛 Waste Collector Dashboard (Mobile only)
Purpose: Collects waste from assigned households and provides feedback.
Features:
1.	Daily Route/Schedule
o	List of households with status:
	Pending ✅
	Collected 🟢
	Missed ❌
2.	Scan QR Code
o	Each household has a QR for easy identification.
3.	Submit Feedback
o	Simple interface:
	Stars for Segregation, Plastic Use
	Checkboxes for common observations
	Remarks (optional)
	Upload photo (optional)
	Deleted automatically if rating > 3.
4.	Compliments/Complaints
o	Can praise or complain about households.
5.	Reason for Missed Pickup
o	Dropdown options (gate locked, no waste, etc.).
6.	Track Collection
o	Daily/weekly status of tasks.
 
🧑‍🌾 Waste Generator Dashboard (Mobile only)
Purpose: Households that generate waste.
Features:
1.	View Collection History
o	Up to 1-month detailed logs of:
	Feedback from collector
	Ratings received
	Compliments or complaints
2.	Provide Feedback
o	Star rating for collectors
o	Remarks/Complaints about them.
3.	Raise Issues
o	Related to garbage dumping, delay in collection, etc.
o	Issues are visible to all generators in the same village.
o	Track manager responses and updates.
4.	Suggestions Box
o	Send suggestions for improvement.
5.	Announcements & Messages
o	Banner section on homepage.
o	View messages sent by manager or admin.
 
🧱 Design/UI Notes
•	Theme: Light green tones, white backgrounds.
•	Buttons:
o	Primary: Light green
o	Danger: Red
o	Info: Blue
•	NavBar: Persistent on top for web, bottom for mobile.
•	Home Sections: Cards with icons and short descriptions.
•	UX:
o	Minimal taps to complete actions
o	Fast submission forms (for collectors)
o	Notifications for new messages/feedback/issues
 
📦 Important Functional Details
•	QR Codes are central: used to track waste collection per household.
•	All data is filtered by villageID, so each village is isolated in logic.
•	MongoDB online used from day one with real-time cloud connection.
•	Issues, reviews, feedbacks, and messages all support reply/update threads.
•	Bulk actions (create users, download QR PDFs) are included.
•	Images are not stored unless necessary (i.e., low rating photos).
